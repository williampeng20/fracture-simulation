In this Project, we will be implementing a system for object collisions and simulating the fractures from these collisions. We will also be using this system to simulate different types of materials and how realistically they will interact with each other.

Problem Description

Watching any modern Hollywood movie, the common destructive effects such as explosions and large-scale collisions become impossible for practical effects to be used. Therefore, the industry has turned to using computer graphics to create these effects. Simulating the collision, fracturing, and destruction of objects will allow us to solve this problem, but the challenge lies in detecting a collision between objects, and then calculating how a collision changes the topology of the objects, even creating multiple sub-objects. This is important as in the real world, when objects collide there is the possibility that one of the objects takes too much force and breaks from impact. This system will allow us to simulate this interaction and model it in the digital space. Building this system will be challenging, as we will need to determine the way and place pressure is accumulated in different materials to model realistic fracturing behavior.

Resources

We will be building our collision and fracture system inside of Blender in order to make use of the rendering framework and animation timeline already built. Blender is built upon Python, and has its own API to import Python scripts. We will be referencing this API here: https://docs.blender.org/api/blender_python_api_2_77_0/info_overview.html
We will be rendering our Blender simulations on our personal computers and desktops, which run on MacOS.

In order to analyze how successful and realistic our system will be, we will attempt to replicate this scene and compare it to our simulation. We will also compare our results to other implementations of simulating fractures, such as voronoi cell fracture and the official Blender fracture modifier, as well as how our rendering time and performance compare up as well.
